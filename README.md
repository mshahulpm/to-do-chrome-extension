## My Todo PWA  

clone this repo 
```shell
git clone https://github.com/mshahulpm/to-do-chrome-extension.git 
```
checkout to branch main
```shell
git checkout pwa 
```
install node modules and build
```shell
yarn && yarn build
```

preview application
```shell
yarn preview
```
go to [http://localhost:4173](http://localhost:4173) and install as pwa
